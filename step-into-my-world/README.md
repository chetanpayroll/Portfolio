A standalone, immersive 60-second experience demonstrating the workflow of a Global Payroll Implementation Manager.

## Overview
This page simulates a real-time work environment featuring:
- **Slack Communication**: handling client queries and internal coordination.
- **Outlook/Email**: managing notifications and priorities.
- **Dashboard**: tracking implementation progress/metrics.
- **Excel/Spreadsheet**: performing data validation and error checking.

## Technical Details
- **Stack**: Pure HTML5, CSS3, ES6 JavaScript (No frameworks).
- **Styling**: Custom CSS with CSS Variables for theming.
- **Fonts**: Inter (UI) and JetBrains Mono (Data).
- **Responsiveness**: Fully responsive (Desktop 2x2 Grid, Tablet 2-row, Mobile Stack).
- **Accessibility**: 
    - Full keyboard navigation support.
    - ARIA labels for screen readers.
    - WCAG AA contrast compliance.
    - Reduced motion support.

## File Structure
- `index.html`: Contains all structure, styles, and logic in a single file for portability.

## Usage
Simply open `index.html` in any modern browser. Click "Begin Experience" to start the 60-second simulation.
